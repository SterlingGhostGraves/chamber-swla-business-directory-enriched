This repository is an independent enrichment and spreadsheet-conversion project based on a publicly derived Chamber Southwest Louisiana business directory summary.

The PDF source reference was not created by this repository owner. It is treated here as original/source reference material for provenance and review.

When adding or changing data, include source links where possible and prefer official business websites, Chamber profile pages, government/public records, or directly confirmed information.
